# Chance

## Описание

Chance — фронтенд-приложение, работающее в экосистеме Telegram как WebApp c интеграцией TON-кошелька. Основная задача: геймификация, управление балансом, подарками, лидербордом, привлечением друзей и другими активностями через красивый реактивный UI. Поддерживаются различные режимы (демо и рабочий), интеграция с бэкендом, поддержка мультиязычности.

## Технологический стек

### Язык
- TypeScript

### Фреймворк и сборка
- React 19 (ReactDOM)
- Vite 6
- Redux Toolkit / React-Redux
- RTK Query (обработка асинхронных API-запросов)
- SCSS (Sass, модульная стилизация)
- Lottie (анимированные элементы)
- Embla Carousel (UI-карусели)
- Classnames (динамические классы)
- Vite Plugins:
  - vite-tsconfig-paths (автозавершение alias импортов)
  - vite-plugin-svgr (SVG как React-компоненты)
  - vite-plugin-node-polyfills (node-полифиллы)
- ESLint (автоматический анализ и контроль качества кода)

### Специфика интеграций
- @tonconnect/ui-react — авторизация и подключение TON-кошелька
- @ton/ton — работа с блокчейном TON (операции, запросы и прочее)
- js-cookie — работа с cookies
- eruda — отладка на мобильных устройствах

### Прочее
- Docker, Dockerfile — для сборки production-образа
- pnpm — менеджер пакетов и сборки

## Основные директории и компоненты

- `src/`
    - `components/` — UI и бизнес-логика (разделены по экранам)
    - `store/` — глобальное состояние на Redux Toolkit, слайсы, api, и RTK Query endpoints
    - `hooks/` — кастомные хуки (например, для работы с Telegram, анимаций, внешних кликов и пр.)
    - `assets/`, `animations/` — графика и анимации проекта
    - `styles/` — глобальные и переменные SCSS
    - `data/` — переводческие json-файлы (en.json, ru.json)
    - `utils/` — вспомогательные функции

## Запуск разработки

1. Убедитесь, что у вас установлен PNPM:
    ```bash
    npm install -g pnpm
    ```
2. Установите зависимости:
    ```bash
    pnpm install
    ```
3. Запустите проект в режиме разработки:
    ```bash
    pnpm dev
    ```
   или
    ```bash
    npm run dev
    ```
4. Откройте приложение по адресу [http://localhost:5173](http://localhost:5173) (или порт, заданный в настройках).


## Дополнительно

- Весь проект строится как одностраничное приложение для WebView Telegram. Не предназначен для статики, только для интеграции с Telegram-клиентом.
- Большинство UI-ресурсов (иконки, анимации, изображения) оптимизированы для загрузки из публичных CDN или локальных ассетов.
- Проект поддерживает мультиязычность (EN и RU, просто расширяется новыми json).
